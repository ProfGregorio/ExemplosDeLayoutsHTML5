# 🎨 Exemplos de Layouts HTML5

Repositório criado para demonstrar diferentes propostas de layouts utilizando **HTML5 Semântico**, servindo como material de apoio para alunos e desenvolvedores que desejam compreender a estruturação de páginas web modernas.

Cada exemplo apresenta uma organização visual diferente, simulando cenários reais encontrados no mercado de desenvolvimento web.

---

## 🌐 Visualização Online

Acesse os exemplos publicados através do GitHub Pages:

👉 https://profgregorio.github.io/ExemplosDeLayoutsHTML5/

---

## 🎯 Objetivos do Projeto

* Praticar HTML5 Semântico;
* Compreender diferentes estruturas de layout;
* Desenvolver boas práticas de organização de páginas;
* Criar bases para aplicação de CSS e JavaScript;
* Servir como referência para projetos futuros.

# 💡 Recomendações para os Alunos

Este projeto tem como foco o estudo da estruturação de páginas utilizando HTML5.

Ao analisar cada layout, procure observar:

* A organização dos elementos;
* A utilização correta das tags semânticas;
* A hierarquia dos conteúdos;
* A disposição visual dos componentes;
* As possibilidades de estilização utilizando CSS.

Além disso, experimente:

* Alterar as cores;
* Criar novas seções;
* Adaptar os layouts para dispositivos móveis;
* Adicionar animações e interatividade com JavaScript.

👉 Valide sempre seus documentos utilizando o W3C Validator:

https://validator.w3.org/
